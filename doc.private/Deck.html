Abstract type representing cards drawn from deck and remaining cards left in the deck.

val cards : res -> Card.t list

cards res returns the cards drawn in res.

val cards_to_string : res -> string list

cards_to_string r returns a string list of the cards drawn in r.

val deck : res -> deck

deck r returns the deck remaining from result res.

val shuffle_deck : unit -> Card.t list

shuffle_deck d is a full 52-card shuffled deck.

val draw_card : Card.t list -> res

draw_card d returns a result of a card drawn from d and the cards left in the deck afterwards.

val draw_two_cards : Card.t list -> res

draw_two_cards d returns a result containing the two cards drawn, and the remaining cards in a separate deck.

val test_deck : (int * int) list -> Card.t list

test_deck c creates a non-randomized deck based off the int in pairs c representing individual cards
