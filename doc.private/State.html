Module State

module State: sig .. end

Abstract value type of state.

type t = {
   deck : Deck.deck;
   dealer : Player.t;
   player : Player.t;
} 
val init_state : int -> t

init_state n is the starting state of a round of black jack, where the dealer and the player have two cards each

val pm : t -> int

pm s is the player's money in state s

val dm : t -> int

dm s is the dealer's money in state s.

val pause : int -> unit

pause s pauses, prints a period, and decrements s until it is 0.

val dealer_draw : t -> t

dealer_draw s is the state s becomes after the dealer draws a card.

val step : t -> Parser.commands -> t

step s cmd is the result of player choice cmd while in state s

val x1_5 : int -> int

x1_5 n multiplies int n by 1.5, then reconverts to int.

val update_money : int -> int

update_money m updates the stats.json file with new money value m.

val calc_money : int -> int -> int

calc_money w b calculates how much the player wins or loses based on bet b and condition w.

val step_round : t -> int -> int -> t

step_round s win bet is a fresh round of blackjack, with the player's money updated according to the result win and wager bet. These are what the possible values of win correspond to: 1: Blackjack victory, 2: Normal Victory, 3: Loss, 4: Draw, 5: Double down victory, 6: Double down loss, Other: Causes a failure

val dealer : t -> Player.t

dealer t returns the dealer from state t.

val player : t -> Player.t

player t returns the player from state t.

val test_state : Deck.deck -> t

test_state d makes a state with a deck input d, for testing.

val get_advice : t -> string

get_advice s gives a string of advice to the player based on s.
