Module Card

module Card: sig .. end

Abstract type representing a card

type t 
val get_val : t -> int

get_val c is the point value of c.

val cards_to_ascii_string : t list -> string

cards_to_ascii_string h turns a card list h into an ASCII string.

val card_facedown_to_ascii_string : t list -> string

card_Facedown_to_ascii_string h turns a card list h into the top card face up and the second card face down, for an input of two cards.

val to_string : t -> string

to_string c is the card type of c as a string.

val make_card : int -> int -> t

make_card f n initializes a card with face value f and the nth time the card has been made, corresponding to one of for suits. Requires: f is in 1..13, n is in 1...4
